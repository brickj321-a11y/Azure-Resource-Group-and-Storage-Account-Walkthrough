## Azure Storage Account File Upload Lab

## Project Summary
This project demonstrates how to use Microsoft Azure to create a resource group, deploy a storage account, upload and edit a file, and safely remove resources to avoid unnecessary costs.

- Project Type: Cloud lab / walkthrough  
- Languages Used: None  
- Environment Used: Azure Portal, local desktop  
- Technologies Used: Azure Resource Groups, Azure Storage Accounts  

## Demonstration

### Step 1: Create Azure Account
Created a free Azure account and logged into the Azure Portal.

### Step 2: Create Resource Group
A resource group was created to organize and manage resources.

### Step 3: Create Storage Account
A storage account was deployed within the resource group.

### Step 4: Create Text File
A plain text file was created locally on the computer.

### Step 5: Upload File
The file was uploaded to the storage account.

### Step 6: Edit File
The file was edited directly within Azure.

### Step 7: Download File
The updated file was downloaded and verified.

### Step 8: Delete Resources
The resource group was deleted to prevent charges.
